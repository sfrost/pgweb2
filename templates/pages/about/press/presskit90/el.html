Πρωτότυπο Δελτίο Τύπου

20 Σεπτεμβρίου 2010: Η Ομάδα Ανάπτυξης PostgreSQL (PostgreSQL Global Development Group) ανακοινώνει την έκδοση 9.0, την πιο πρόσφατη έκδοση της πρωτοπόρου ανοιχτού κώδικα σχεσιακής βάσης δεδομένων. Σε αυτή την έκδοση περιλαμβάνονται βελτιώσεις και προσθήκες σε ασφάλεια, υποστήριξη εφαρμογών, monitoring, επιδόσεων και υποστήριξη αποθήκευσης ειδικής μορφής δεδομένων. Είναι ιδιαίτερα σημαντικές οι νέες προσθήκες που υποστηρίζουν λειτουργίες replication και hot standby, επιταχύνοντας την αποδοχή της PostgeSQL σε περιβάλλοντα Cloud και scalable εφαρμογών.

"Η Ενσωματωμένη υποστήριξη για replication και hot standby στην 9.0 αποτελούσαν τα πιο δημοφιλή αιτήματα για αρκετά χρόνια." είπε ο Simon Riggs, CTO στην 2ndQuadrant και contributor στο Hot standby της PostgreSQL. "Το Αποτελεσματικό, low latency streaming replication προστατεύει καλύτερα τα δεδομένα σας, ενώ το hot standby μειώνει σημαντικά το συνολικό κόστος λειτουργίας. Σε proprietary προϊόντα, αυτή η δυνατότητα είναι μια προσθήκη που κοστίζει χιλιάδες δολάρια."

H Έκδοση 9.0 περιλαμβάνει αρκετές (πάνω από 10) σημαντικές βελτιώσεις σε κάθε πτυχή της σχεδίασης και των επιδόσεων, συμπεριλαμβανομένων: υποστήριξη 64-bit Windows, conditional and per-column triggers, upgrade-in-place from 8.3 and 8.4, deferrable unique constraints and high-performance event messaging. Για μια πλήρη λίστα των νέων χαρακτηριστικών, δείτε τη σελίδα μας με τις πληροφορίες Έκδοσης.

"Ως ένας απο τους παγκόσμιος μεγαλύτερους διαχειριστές υποδομών Internet registries, Η Afilias εκφράζει την ικανοποίηση της για την ενσωμάτωση in-place upgrades στην PostgreSQL 9.0, αυτο το χαρακτηριστικό μας επιτρέπει να αναβαθμίσουμε τις 16 εκατομμύρια domain name βάσεις μας με την ελάχιστη ζημία", δήλωσε ο Ram Mohan CTO στην Afilias.

"Είμαστε πολύ ενθουσιασμένοι με τα νέα χαρακτηριστικά ... ιδίως με τα deferrable unique constraints ως μια ευκαιρία για την επιτάχυνση των διαδικασιών update και upgrade για τους πελάτες μας" δήλωσε ο Paolo Juvara, CTO της OpenBravo.

Ο Marc Balmer, ιδρυτής της Micro Systems στην Ελβετία, ενδιαφέρεται περισσότερο για την βελτιώση της λειτουργίας LISTEN/NOTIFY event messaging: "Το καινούριο option data payload στα ασύγχρονα notifications θα ενισχύσει την επόμενη έκδοση τον εφαρμογών εμφάνισης απομακρυσμένων δεδομένων που αναπτύσσουμε."

Χάρη στη συμβολή προγραμματιστών και επιχειρήσεων απο όλο τον κόσμο, όπως οι NTT Open Source, EnterpriseDB, 2ndQuadrant, Red Hat και πολλών άλλων, o ρυθμός ανάπτυξης και υιοθέτησης από χρήστες είναι μεγαλύτερος απο οποτεδήποτε.

«Εδώ στην τράπεζα CAIXA χρησιμοποιούμε PostgreSQL σε περιβάλλοντα διαχείρισης κρίσιμων οικονομικών δεδομένων γιατί έχει την δυνατότητα να υποστηρίξει τις λειτουργίες μας", δήλωσε ο Clarice Coppetti, IT Αντιπρόεδρος της CAIXA Τράπεζας στην Βραζιλία. "Η νέα έκδοση δείχνει ότι η PostgreSQL αναπτύσσεται γρήγορα με τρόπο που την κάνει να μπορεί να ανταποκρίνεται στις ανάγκες της επιχείρησής μας."

Σχετικά με την PostgreSQL: Η PostgreSQL είναι η πρώτη ανοιχτού λογισμικού βάση δεδομένων, με μια παγκόσμια κοινότητα χιλιάδων χρηστών και contributors καθως και δεκάδες εταιρείες και οργανισμούς. Το Project της PostgreSQL διαθέτει πάνω απο 20 χρόνια ανάπτυξης, ξεκίνησε απο το Πανεπιστήμιο της Καλιφόρνιας, Berkeley, και σήμερα ο ρυθμός ανάπτυξης του είναι χωρίς προηγούμενο. Τα χαρακτηριστικά της PostgreSQL οχι μόνο ισοτιμούν με αντίστοιχα κορυφαίων proprietary συστημάτων βάσεων δεδομένων, αλλά υπερβαίνουν αυτά σε προηγμένα χαρακτηριστικά όπως επεκτασιμότητα, ασφάλεια και σταθερότητα. Μπορείτε να μάθετε περισσότερα για την PostgreSQL και να συμμετέχετε στην κοινότητά μας, στο www.postgresql.org.

Αναλυτική πληροφόρηση σχετικά με την Έκδοση

Feature List

Η Έκδοση 9.0 έχει ένα πρωτοφανή αριθμό νέων σημαντικών χαρακτηριστικών, που καταδεικνύουν την επιτάχυνση της ανάπτυξης της κοινότητας προγραμματιστών και υποστηρικτών μας κατά το τελευταίο έτος:

Replication and Scalability

  • Hot Standby
  • Streaming Replication

Διαχείριση και Ασφάλεια

  • Integrated Upgrade-in-Place (pg_upgrade)
  • RADIUS Authentication
  • Password Strength Checking (passwordcheck)
  • Easier Database Permissions Management Commands (GRANT ON ALL and DEFAULT PERMISSIONS)

Σχεδιασμός Βάσης και SQL

  • Deferrable Unique Constraints
  • Conditional Triggers
  • Column Triggers
  • Ordering in Aggregates
  • New Windowing Functions (ROWS PRECEDING and FOLLOWING)

Stored Procedures

  • Anonymous Procedure Code Blocks (DO statement)
  • Improved Perl and Python Stored Procedures (including support for Python 3)
  • Named Parameters in Stored Procedure Calls

Επιδόσεις και προηγμένα χαρακτηριστικά

  • Improved Event Messaging (LISTEN/NOTIFY)
  • 64-bit Windows Support
  • Optimization for ORM-Generated Queries (JOIN removal)
  • Unique Keys for Non-Scalar Data (Exclusion Constraints)
  • Expanded Support for Key-value Data (HStore)
  • JSON and XML Explain Plans

Άδεια χρήσης

Η PostgreSQL χρησιμοποιεί την άδεια χρήσης PostgreSQL, μια παρόμοια με BSD άδεια που προϋποθέτει μόνο τη διατήρηση των στοιχείων πνευματικών δεδομένων και άδειας χρήσης του πηγαίου κώδικα που αφορά. Αυτή η πιστοποιημένη κατά OSI άδεια χρήσης αξιολογείται ευρέος ως ευέλικτη και φιλική προς τις επιχειρήσεις, καθώς δεν περιορίζει τη χρήση της PostgreSQL με εμπορικές και αποκλειστικές εφαρμογές. Μαζί με την υποστήριξη πολλών εταιριών και τη δημόσια ιδιοκτησία του κώδικα, η άδεια χρήσης BSD καθιστά την PostgreSQL πολύ δημοφιλή στους προμηθευτές που θέλουν να ενσωματώσουν μια βάση δεδομένων στο δικό τους προϊόν χωρίς να φοβούνται τις χρεώσεις, το vendor lock-in ή τις αλλαγές στους όρους της Άδειας χρήσης.

Εταιρική Υποστήριξη
